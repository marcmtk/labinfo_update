# Bakterielt luftvejspanel DNA

| IUPAC: - | **Bakterielt luftvejspanel DNA** |
|---|---|
| ![DANAK mærke](formularer/Upload/2022/10/DANAK_maerke_1.png) | |

| | |
|---|---|
| **UNDERSØGELSENS PRINCIP** | Påvisning af specifikke gensekvenser fra *Bordetella parapertusis* (BPP), *Bordetella pertusis* (BP), *Chlamydophila pneumoniae* (CP), *Mycoplasma pneumoniae* (MP), *Legionella pneumophila* (LP), *Streptococcus pneumoniae* (SP) og *Haemophilus influenzae* (HI) ved hjælp af multiplex Real-Time PCR (Allplex™). |
| **INDIKATION** | Mistanke om pneumoni forårsaget af bakterier. |
| **PRØVEMATERIALE** | Nasotrakealsekret, nasopharyngeal podeprøve, oropharyngeal podeprøve, bronchoalveolar lavage (BAL), ekspektorat og trachealsekret.<br><br>Fra praksis analyseres kun prøver fra nedre luftveje.<br>Podeprøver undersøges kun såfremt der er gjort relevant forsøg på udtagning af materiale fra nedre luftveje først. |
| **UTENSILIER** | COPAN E-Swab eller UTM. Rør (sterilt tør glas) fra trakealsugesæt eller ekspektoratdobbeltdåse. |
| **PRØVETAGNING** | Så tidligt som muligt i sygdomsforløbet. |
| **OPBEVARING** | Prøvematerialet opbevares ved 2-8°C i op til 3 dage (fra prøvetagningstidspunkt til analyse). |
| **FORSENDELSE** | Prøven bestilles i BCC Mikrobiologi, analyse nummer 220. Bringes straks til laboratoriet. |
| **SVARAFGIVELSE** | Svar afgives som henholdsvis "detected/positiv", "not detected/negativ", "invalid", (som svares "Prøvematerialet indeholder stoffer, som hæmmer PCR-undersøgelsen. Ny prøve udbedes") eller analysefejl (som svares "Analysen er desværre mislykket. Vi kører prøven om igen, så send ikke en ny prøve. Nyt svar følger – ring venligst ikke til KMA").<br><br>Der kan forventes svar indenfor 2 hverdage. |
| **TOLKNING** | Et positiv PCR-svar er forenelig med aktuel eller nylig infektion. PCR metoden påviser levende såvel som døde bakterier og kan derfor ikke anvendes til behandlingskontrol. Et negativt resultat udelukker ikke infektion med de undersøgte bakterier. |
| **SÆRLIGE BEMÆRKNINGER** | Ved mistanke om infektion med Legionella pneumophila skal der sendes materiale fra nedre luftveje (BAL, trakealsekret eller ekspektorat). |
